## 作用
- **修饰变量**：静态变量，属于类，全局唯一，类加载时初始化
- **修饰方法**：静态方法，属于类，可通过类名直接调用，无this指针
- **修饰代码块**：静态代码块，类加载时执行一次，用于初始化静态资源
- **修饰内部类**：静态内部类，不依赖外部类对象，可直接创建

## static 方法可以调用非 static 方法吗
静态方法不能直接调用非静态方法/变量，因为非静态成员属于对象，需先创建对象才能调用；非静态方法可直接调用静态成员。
