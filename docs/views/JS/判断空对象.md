## 使用ES6的Object.keys()方法
```javascript
    let obj = {}
    let arr = Object.keys(obj)
    console.log(arr.length === 0) //true 为空，false 不为空
```

## 将json对象转化为json字符串，再判断该字符串是否为"{}"
```javascript
    let obj = {}
    let b = (JSON.stringify(obj) === "{}")
    console.log(b)  //true 为空，false 不为空
```
