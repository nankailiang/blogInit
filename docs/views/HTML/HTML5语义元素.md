- 语义元素是拥有语义的元素。
- 语义元素清楚地向浏览器和开发者描述其意义。
- 非语义元素的例子：`<div>` 和 `<span>` - 无法提供关于其内容的信息。
- 语义元素的例子：`<form>`、`<table>` 以及 `<img>` - 清晰地定义其内容。

```html
// 常见的语义化标签
<article>
<aside>
<details>
<figcaption>
<figure>
<footer>
<header>
<main>
<mark>
<nav>
<section>
<summary>
<time>
```
